# data-cleaning-and-scraping

This project uses the possible link between CrossFit and weightlifting to check an hypothesis.

## Objective
To determine whether the rise of CrossFit in Europe is sufficient to justify opening specialized weightlifting centers.

## Dataset
- Source:     
    For cleaning:     
        - https://es.wikipedia.org/wiki/Campeonato_Europeo_de_Halterofilia_de_2019
        - https://es.wikipedia.org/wiki/Campeonato_Europeo_de_Halterofilia_de_2021 (2020 data not present due to COVID)

    For scraping: 
        -https://es.wikipedia.org/wiki/Campeonato_Europeo_de_Halterofilia_de_2024

- Contents: raw and processed data

## Methodology
- Data cleaning
- Handling text
- DType conversion
- Fix missing values
- Web scraping
- Exploratory Data Analysis

## Deliverables
- Jupyter Notebook with full analysis

## How to Use
Open the notebook in Jupyter to reproduce analysis.
